# Geomagic

Plugin for working with Geo locations in serverside.
Includes:

* geokit-rails3
* geocoder
* GeoRuby
* spatial_adapter

## Installation

Add this line to your application's Gemfile:

    gem 'geomagic'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install geomagic
